# ariesh-git.github.io

Personal portfolio site for **Ariesh K** — DevOps &amp; Cloud Engineer.

🌐 Live: [https://ariesh-git.github.io](https://ariesh-git.github.io)

## Stack

Vanilla HTML, CSS, and JavaScript — no build step, no framework.
Hosted for free on GitHub Pages.

## Local preview

```bash
# any static server works; here's one option
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deploy

1. Push to the `main` branch of the `ariesh-git.github.io` repository.
2. In the repo → **Settings → Pages**, set **Source** to `main` / root.
3. Site publishes at `https://ariesh-git.github.io` within ~1 minute.

## Structure

```
├── index.html      # markup + content
├── styles.css      # design tokens + all styling
├── script.js       # typewriter, scroll reveal, nav highlight

```
